APU Scholars Hub is a student-led academic website designed to support collaboration, resource sharing, and engagement among the Asia Pacific University academic community.

To provide a centralized digital platform that promotes academic activities, peer learning, event sharing, and access to academic materials for APU students and staff.

The website was developed using HTML, CSS, and JavaScript. Wireframes were created using design tools, and GitHub was used for version control.

Pages include Home, About Us, Events, Resources, Blogs, Gallery, Members, FAQ, Links, and Contact.

Target users include current APU students, alumni, academic staff, recruiters, and external visitors interested in academic initiatives.

It allows students to explore opportunities, download learning resources, view past and upcoming events, and connect with peers and mentors.

Some challenges included CSS layout consistency, responsive design on mobile, and managing multi-page content. These were addressed through team reviews and thorough testing.

Not yet, but future updates may include member login features for submitting blogs, feedback, and event suggestions.

Planned improvements include search functionality, admin control panels, dynamic event pages, and student dashboards.

Shelton handled Home, About, Events, and Gallery. Mustafa developed Members, Links, and FAQ. Joyce prepared documentation. Ethan worked on Blogs and Contacts.
